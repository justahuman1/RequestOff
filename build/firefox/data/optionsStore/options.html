Auto-Timer Settings

An countdown timer that automatically adds tabs to offline mode if they are inactive.

Action Value
Enable Timer:
Evict Time:
The threshold minimum (seconds) to turn the requests for the tab off. Ex: 90 means that any tab that has not been visited in >90 seconds will be evicted to request off.
Tab Poll Time:
The interval (seconds) to check the last time a tab has been visited. The larger the number, the smaller the extension's footprint but also increases the standard deviation of evicting a tab to request off.
